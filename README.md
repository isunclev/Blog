# Blog
DJango+Bootstrap开发博客系统
</hr>
<h2>项目功能点</h2>
</hr>
<ul>
  <li>Blog应用开发</li>
  <ul>
    <li>博客应用设计</li>
    <li>博客列表分页</li>
    <li>获取当前博客的上一篇和下一篇博客</li>
    <li>博客内容分类和分类统计</li>
    <li>博客点赞</li>
    <li>博客评论</li>
    <li>用户注册/登录/修改密码</li>
  </ul>
</ul>
